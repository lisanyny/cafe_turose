# cafe_Turose

## サイト概要
### サイトテーマ
カフェの来店予約とオンライン販売の両方ができるサイト

### テーマを選んだ理由
自身のお店を出す夢を持っている親戚が、実際に出店する際に少しでも役に立てばと思いサイトを作成しようと考えました。

## 開発環境
- OS：Amazon Linux 2
- 言語：HTML,CSS,JavaScript,Ruby,SQL
- フレームワーク：Ruby on Rails
- JSライブラリ：jQuery
- IDE：Cloud9
